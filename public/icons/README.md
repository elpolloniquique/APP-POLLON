# Iconos PWA — El Pollón

Coloca aquí los iconos oficiales de la app instalable.

| Archivo | Tamaño | Uso |
|---------|--------|-----|
| `icon-192.png` | 192×192 px | Android, accesos directos |
| `icon-512.png` | 512×512 px | Splash, instalación |
| `icon-maskable-192.png` | 192×192 px | Android adaptive (zona segura ~80%) |
| `icon-maskable-512.png` | 512×512 px | Android adaptive grande |

**Origen recomendado:** exportar desde `public/img/logo pollon.png` o `logo.png`.

**Herramientas gratuitas:**
- [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator)
- [Maskable.app](https://maskable.app/) para iconos maskable

Por ahora se usan copias del logo. Reemplázalos por PNG optimizados antes de producción final.
